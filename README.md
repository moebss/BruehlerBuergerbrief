# Brühler Bürgerbrief

Digitale Bürgerinformations-Plattform für die Stadt Brühl.

## Über das Projekt
Dieses Projekt ermöglicht Bürgerinnen und Bürgern der Stadt Brühl, ihre Anliegen, Ideen und Verbesserungsvorschläge direkt an die lokale Politik zu übermitteln.

## Features
- **Interaktives Formular:** Schritt-für-Schritt Erfassung von Anliegen.
- **Lokalisierung:** Speziell angepasst auf Brühler Stadtteile (Badorf, Kierberg, Heide, etc.).
- **Responsive Design:** Optimiert für Desktop und Mobile.
- **Datenschutz:** DSGVO-konforme Verarbeitung.

## Technologie
- HTML5, CSS3 (Vanilla), JavaScript
- Keine externen Framework-Abhängigkeiten
- Hosting: Vercel (empfohlen)

## Setup
1. Repository klonen
2. `index.html` im Browser öffnen oder via Live Server starten.

## Anpassung
Die Datei `index.html` enthält alle relevanten Inhalte.
Die Formular-Übermittlung nutzt Formspree (ID muss in `action="..."` eingetragen werden).
